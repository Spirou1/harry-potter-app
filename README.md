## Harry Potter API 

O objetivo deste trabalho é consultar a **HP-API** para demonstrar os conhecimentos adquiridos em **corrotinas** e **web services** . Uma rápida utilização dos métodos oferecidos pela API, usando a interface Web disponível, permite que o desenvolvedor entenda como consumir os endpoints disponíveis.

## Listar Personagem Específico 

Permite ao usuário informar um **ID** para busca via endpoint da API. A resposta exibe na tela: 

name 

species house Foto do personagem 

## Listar Professor da Escola 

Permite ao usuário informar um **nome de professor** para busca via endpoint da API. A resposta exibe na tela: 

name alternate_names species house 

## Listar Estudantes de uma Casa 

Exibe o nome de todos os estudantes de uma casa selecionada via **Radio Buttons** . A resposta é mostrada em um **RecyclerView** com: 

Foto em tamanho pequeno Nome do estudante Casa do estudante 

Não é necessário capturar o clique nas células do RecyclerView. 

## Ver Feitiços 

Lista todos os feitiços da API em uma **lista ou RecyclerView** . Ao clicar em um feitiço, abre tela de detalhes com: name 

description
